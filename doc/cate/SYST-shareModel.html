SYST.shareModel(共享数据模型)

models Object

模型集合

add Function (key, model)

  • key: 键值，获取时标识
  • model: SYST.Model对象

添加 model

实例：


    var userModel = SYST.Model({ name: '张三' });
    SYST.shareModel.add('userModel', userModel);

get Function (key) => SYST.Model

  • key: 键值，标识

获取 model

实例：


    SYST.shareModel.get('userModel');

remove Function (key)

  • key: 键值，标识

移除 model

实例：


    SYST.shareModel.remove('userModel');

has Function (key) => Boolean

  • key: 键值，标识

判断某个model是否存在

实例：


    var userModel = SYST.Model({ name: '张三' });
    SYST.shareModel.add('userModel', userModel);
    SYST.shareModel.has('userModel'); //=> true
