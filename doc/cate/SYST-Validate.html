SYST.Validate(校验函数)

isEmpty Functon (variable)=> Boolean

判断变量是否为空。
!variable、variable.length = 0、''、null、undefined => true

isCN Functon (val, [flag])=> Boolean

  • val: 待判定的值
  • flag: 默认 false, true为完全匹配

判断当前值是否含有中文字符。

实例：


    SYST.V.isCN('0as中df');
    //=> true
    SYST.V.isCN('0as国df', true);
    //=> false
