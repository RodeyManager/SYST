SYST

SYST 是一个简单的 JavaScript mvc framework

SYST包含了前端开发简单的mvc分离模式，但是没有双向绑定功能（功能待善...）。 初意在于将前端js代码更好的分离，其中很大一部分是在事件处理和数据处理上。 其中包含了一些前端开发常用的方法。依赖于jQuery或Zepto。

下载 SYST

  • SYST.js v2.0.4 (开发板)
  • SYST.min.js v2.0.4 (生产版)
  • Source code on Github

引入SYST前必须先引入jQuery或者Zepto
