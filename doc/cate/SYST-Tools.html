SYST.Tools(常用方法)

parseDom Function (htmlStr, [tagPanel])

  • htmlStr: 元素字符串 String
  • tagPanel: 包装的标签名称

将元素转成对象并返回

实例：


    var elm = SYST.View().parseDom('<p id="intro" >8899</p>');
    console.log(elm); //=> HTMLElement
    console.log(elm.id) //=> intro

trigger Function (element, event, data, callback)

  • element: HTMLElment对象
  • event: 事件名称
  • data: 待传输的数据
  • callback: 待处理的事件函数

侦听事件并传递数据

实例：


    SYST.T.trigger(document.body, 'onclick', { name: 'body' }, function(evt, data){
        console.log(this); //=>:  <body>...</body>
        console.log(evt); //=>:  MouseEvent
        console.log(data); //=>   { name: 'body' }
    });

trim Function (str)

  • str: 字符串

实例：


    var name = SYST.T.trim(' I name is Jack ');
    console.log(name); //=> I name is Jack

rtrim Function (str, commer)

  • str: 字符串
  • commer: 待清楚的字符

清楚制定字符

实例：


    var name = SYST.T.rtrim('# I name is Jack #', '#');
    console.log(name); //=> I name is Jack

indexOf Function (array, value)

  • array: 目标数组对象
  • value: 制定的元素

检查制定元素在目标数组中的索引

实例：


    var index = SYST.T.indexOf(['b', 'a', 1], 'a');
    console.log(index); //=> 1

arrRemove Function (array, value)

  • array: 目标数组对象
  • value: 制定待删除的元素

从数组中删除指定的元素, 返回被指定的元素（ 执行后原数值array将移除指定的元素 ）, 返回删除后的数组。

实例：


    var value = SYST.T.arrRemove(['b', 'a', 1], 'a');
    console.log(value); //=> ['b', 1]

F2C Function (str)

  • str: 字符串

全角字符转为半角,并去除所有空格。

实例：


    var value = SYST.T.F2C(' 中国人民共和　国 ');
    console.log(value); //=> 中国人民共和国

toFirstUpperCase Function (str)

  • str: 字符串

首字符大写

escapeHtml Function (htmlStr)

  • str: 字符串

html字符转义。

实例：


    var value = SYST.T.escapeHtml('<div id="main"></div>');
    console.log(value); //=> &lt;div id=&quot;main&quot;&gt;&lt;/div&gt;

dateFm Function (n)

  • n: 数字

补0。

实例：


    var value = SYST.T.dateFm(6);
    console.log(value); //=> 06;

setDateFormat Function (timestamp, [format], [prefix])

  • timestamp: 时间戳
  • format: 显示格式 如：'Y-m-d h:i:s'
  • prefix: 是否补0

格式化时间戳

实例：


    var date = SYST.T.setDateFormat(Date.now(), 'Y-m-d h:i:s');
    console.log(date); //=> 2016-4-21 14:3:1

    //======>
    var date = SYST.T.setDateFormat(Date.now(), 'Y-m-d');
    console.log(date); //=> 2016-4-21

    //=====>
    var date = SYST.T.setDateFormat(Date.now(), 'm-d h:i:s');
    console.log(date); //=> 4-21 14:4:3

    //=====>
    var date = SYST.T.setDateFormat(Date.now(), 'Y-m-d h:i:s', true);
    console.log(date); //=> 2016-04-21 14:04:49

setDateFormatCN Function (timestamp, [format], [prefix])

  • timestamp: 时间戳
  • format: 显示格式 如：'Y-m-d h:i:s'
  • prefix: 是否补0

格式化时间戳，转为中文时间格式，如："2016年5月4日 10时03分20秒"

实例：


    var date = SYST.T.setDateFormatCN('2016/5/4 10:03:20');
    console.log(date); //=> 2016年5月4日 10时03分20秒

    var date = SYST.T.setDateFormatCN(Date.now(), 'YY-mm-dd h:i:s');
    console.log(date); //=> 16年5月4日 10时11分5秒

getDateDiff Function (dateOne, dateTwo)

  • DateOne: 开始时间
  • DateTwo: 结束时间

比较两个时间差，返回一个对象包含日时分秒

实例：


    var diff = SYST.T.getDateDiff('2015-06-06 05:20:02', '2015-06-01 10:10:10');
    console.log(diff); //=> Object {days: 361, hours: 4, minutes: 50, seconds: 8}

randomChar Function (size, [flag])

  • size: 生成的随机字符长度
  • flag: 是否去除干扰，默认为 true; 去除扰乱视线 I L l o O

获取随机字符（包括数字和字母）

实例：


    var chars = SYST.T.randomChar(15);
    console.log(chars); //=> RrZgBLyE0m0wAB8

params Function ([name, url])

  • name: 指定参数名
  • url: 指定需要的目标url

获取指定参数或者所有参数列表

实例：


    /*如当前url为：http://localhost/cmbx/mall/build/insure.html#!step1/:10000?id=10&name=javascript&mvc=react*/
    var params = SYST.T.params();
    console.log(params); //=> Object {id: "10", name: "javascript", mvc: "react"}

    name = SYST.T.params('name');
    console.log(name); //=> javascript

getParams Function (name, [url]) //=> String

  • name: 指定参数名
  • url: 指定需要的目标url

获取get模式下url中的指定参数值

实例：


    /*如当前url为：http://localhost/cmbx/mall/build/insure.html#!step1/:10000?id=10&name=javascript&mvc=react*/
    var name = SYST.T.getParams('name');
    console.log(name); //=> javascript

paramData Function (data, [flag]) //=> String

  • data: Object 参数列表
  • flag: 指定是否在生成后的字符串前面添加 ?

格式化参数 flag: 表示前面是否加上‘?’返回，true: 加上；false: 不加(默认)

实例：


    var data = { id: 10, name: 'javascript', mvc: 'react' };
    console.log(SYST.T.paramData(data, true)); //=> ?id=10&name=javascript&mvc=react

serialize Function (data, [flag]) //=> String

将对象序列化为字符串

实例：


    SYST.T.serialize({ name: 'Tom', age: 98 });
    /* => "name=Tom&age=98" */

unserialize Function (str) //=> object

将字符串反序列化为对象

实例：


    SYST.T.unserialize("name=Jack&age=45&d=0");
    // => Object {name: "Jack", age: "45", d: "0"}
    SYST.T.unserialize("http://www.baidu.com?name=Jack&age=45&d=0");
    // => Object {name: "Jack", age: "45", d: "0"}

jumpTo Function (url, [params])

  • url: 需要跳转的 url 地址
  • params: 跳转时url上面需要附加的参数

跳转，可以指定参数

实例：


    SYST.T.jumpTo('https://www.baidu.com/s', { wd: 'javascript' });

redirect Function (url, [params])

jumpTo别名

displace Function (str, [...args])

  • str: 组装的初始字符串
  • args: 替换列表

组装字符串

实例：


    var url = SYST.T.displace('https://%s/s', 'www.baidu.com');
    console.log(url); //=> https://www.baidu.com/s

    var intro = SYST.T.displace('我的名字叫%s, 我今年%d岁。', '张三', 23);
    console.log(intro); //=> 我的名字叫张三, 我今年23岁。

Cookie Function (key, value, [options])

  • key: cookie键值
  • value: 需要保存的值
  • options: 其他可选项
    • expires: 设置cookie过期时间，Number，单位以天计算
    • domain: 设置可访问的主域

浏览器 cookie操作

实例：


    //设置 cookie
    SYST.T.Cookie('name', '张三');
    //获取 cookie
    var name = SYST.T.Cookie('name');
    console.log(name); //=> 张三
    //过期时间, 保留20天
    SYST.T.Cookie('username', 'zhangsan', { expires: 20 });
    //指定可访问域
    SYST.T.Cookie('username', 'zhangsan', { domain: 'www.senuu.com' });

getCookie Function (key)

  • key: cookie键值 String or Array

浏览器 cookie操作，获取指定的cookie

实例：


    //获取 cookie
    var name = SYST.T.getCookie('name');
    console.log(name); //=> 张三
    //获取多个
    var cookies = SYST.T.getCookie(['name', 'age']);
    //=> Object { name: '张三', age: 29 }

setCookie Function (keys, value, [options])

  • keys: 数据 String or Object
  • value: 数据 String
  • options: 其他可选项

浏览器 cookie操作，设置cookie

实例：


    SYST.T.setCookie('name', 'Rodey');
    SYST.T.setCookie({ name: 'Rodey', age: 128 });

each Function (object, callback)

  • object: Array数组或者Object对象
  • callback: 遍历回调函数

遍历对象

实例：


    var array = [1, 'a', { name: 'b' }, ['张三', '李四']];
    SYST.T.each(array, function(item, index){
        console.log(item, index);
        //=> 1 0
        //=> a 1
        //=> Object {name: "b"} 2
        //=> ["张三", "李四"] 3
    });

    var object = { name: 'b', age: 23, profession: 'Web前端开发工程师' };
    SYST.T.each(object, function(item, index, key){
        console.log(key + ': ' + item, 'index: ' + index);
        //=> name: b index: 0
        //=> age: 23 index: 1
        //=> profession: Web前端开发工程师 index: 2
    });

render Function (tpl, data, [helper])

  • tpl: 模板字符串或包含模板元素的id
  • data: 数据对象
  • helper: 自定义辅助函数

SYST.tplConfig 设置 模板开关标签, 默认为: { open: '<%', close: '%>' }

模板渲染

实例：

one: 

    var html = SYST.T.render('<h1><%= title %></h1>', { title: 'View Template' });

            %>
            <li>Name：<%= user.name %>      Age：<%= user.age %></li>
            <li><%= user.age %></li>
            <% } } %>
        </ul>

    var data = {
        title: '用户列表',
