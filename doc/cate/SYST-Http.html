SYST.Http(请求相关)

load Function (dom, url, [data], [callback])

  • dom: 载入页面成功后存放的dom节点
  • url: 待载入页面的URL地址
  • data: 待发送 Key/value 参数。
  • callback: 载入成功时回调函数。

载入远程 HTML 文件代码并插入至 DOM 中。
默认使用 GET 方式 - 传递附加参数时自动转换为 POST 方式。 可以指定选择符，来筛选载入的 HTML 文档，DOM 中将仅插入筛选出的 HTML 代码。 语法形如 "url #some > selector"。请查看示例。

实例：


    SYST.Http().load("body", "feeds.php", {limit: 25}, function(res){
        alert("载入成功");
    });

get Function (url, [data], [callback], [type])

  • url: 待载入页面的URL地址
  • data: 待发送 Key/value 参数。
  • callback: 载入成功时回调函数。
  • type: 返回内容格式，xml, html, script, json, text, _default。

通过远程 HTTP GET 请求载入信息。
这是一个简单的 GET 请求功能以取代复杂 $.ajax。 请求成功时可调用回调函数。如果需要在出错时执行函数，请使用 $.ajax。

实例：


    SYST.Http().get("test.do", { name: "John", time: "2pm" }, function(res){
        alert("Data Loaded: " + res);
    });

getJSON Function (url, [data], [callback])

  • url: 待载入页面的URL地址
  • data: 待发送 Key/value 参数。
  • callback: 载入成功时回调函数。

通过 HTTP GET 请求载入 JSON 数据。
您可以通过使用JSONP形式的回调函数来加载其他网域的JSON数据， 如 "myurl?callback=?"。将自动替换 ? 为正确的函数名，以执行回调函数。 注意：此行以后的代码将在这个回调函数执行前执行。

实例：


    SYST.Http().getJSON("test.do", { name: "John", time: "2pm" }, function(res){
        alert("Data Loaded: " + res);
    });

getScript Function (url, [callback])

  • url: 待载入页面的URL地址
  • callback: 载入成功时回调函数。

通过 HTTP GET 请求载入并执行一个 JavaScript 文件。
注意：如果通过 getScript 加入脚本，请加入延时函数。

实例：


    SYST.Http().getScript("test.js", function(){
        alert("Script loaded and executed.");
    });

  • url: 待载入页面的URL地址

通过远程 HTTP POST 请求载入信息。

doRequest Function (url, data, success, error, [options])

  • url: 接口地址
  • data: 待发送 Key/value 参数。
  • success: 载入成功时回调函数。
  • error: 载入失败时回调函数。
  • type: 返回内容格式，xml, html, script, json, text, _default。

通过Ajax获取请求接口，获取信息。

实例：


    SYST.Http().doRequest("test.php", { name: "John", time: "2pm" }, function(res){
        alert("Data Loaded: " + res);
    });
