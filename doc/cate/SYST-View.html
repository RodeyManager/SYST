SYST.View(视图)

container String

视图容器，支持jquery选择器

实例：

html: 

    <body>
        <div id="header"></div>
        <div id="main">
            <div id="aside">aside</div>
        </div>
        <div id="footer"></div>
    </body>
javascript: 

    SYST.View({
        container: '#main',
        events: {
            'click #aside'    : 'clickAside'
        },
        clickHandler: function(evt){
            console.log(evt.currentTarget, evt.target);
        }
    });

destroy Function

移除视图容器中的所有节点（将视图容器置为空）

hoadEvent Function (target, func)

  • target: 目标对象
  • func: 处理函数

改变对象属性作用域 (常用在元素触发事件侦听函数中)

实例：


    var obj = { name: 'a'};
    document.body.addEventListener('click', SYST.View().hoadEvent(obj, function(evt){
        console.log(evt.target);
        console.log(this) //=> obj
    }), false);

onEvent Function (selector, event, func)

  • selector: 元素选取器
  • event: 事件名称 String，如：'click'、'mouseover'等
  • func: 事件处理函数

元素触发事件侦听

实例：

one: 

    var view = SYST.View();
    function clickHandler(evt){
        console.log(evt);
    }
    view.onEvent('body', 'click', clickHandler);
two: 

    var view = SYST.View({
        init: function(){
            this.onEvent('body', 'click', this.clickHandler);
        },
        //事件处理
        clickHandler: function(evt){
            console.log(evt.currentTarget || evt.target);
            console.log(this);
        }
    });

offEvent Function (selector, event, func)

  • selector: 元素选取器
  • event: 事件名称 String，如：'click'、'mouseover'等
  • func: 事件处理函数

元素移除事件侦听函数

实例：


    var view = SYST.View();
    function clickHandler(evt){
        console.log(evt);
    }
    view.offEvent('body', 'click', clickHandler);

template Function (tpl, data, [helper])

  • tpl: 模板字符串或包含模板元素的id
  • data: 数据对象
  • helper: 自定义辅助函数

模板渲染 请参考 SYST.T.render

parseDom Function (htmlStr, [tagPanel])

参考 SYST.T.parseDom

实例：


    var elm = SYST.View().parseDom('<p>8899</p>');
    console.log(elm); //=> HTMLElement

model SYST.View

默认View将关联一个SYST Model对象，你可以在构建View对象时进行覆盖

实例：


    var view = SYST.View();
    console.log(view.model); //=> SYST.Model

    //===========>
    var model = SYST.Model({ name: 'view->model' });
    var view = SYST.View({
        model: model
    });
    console.log(view.model.name); //=> view-model
